Установка:

1. Измените данные в файле config.php.
2. Выставьте права на запись файлу /lib/local.xml
3. Запустите файл cron.php для обновления тарифной сетки.
	Для изменения порядка вывода стран укажите двухбуквенные коды стран в файле dropdown.js,
	в массиве first_countries. Например first_countries = Array('RU','UA','IL');
	
Пример носит демонстративный характер и без доработки не может быть использован.
